# Data Analysis API

A backend data analysis service built using **FastAPI, Pandas, and NumPy** to process datasets and generate analytical insights through REST APIs.

The project aims to provide endpoints that allow users to upload datasets, explore data statistics, and retrieve insights programmatically.

---

## Features

- Dataset summary statistics
- Column exploration
- Data aggregation
- REST API endpoints for data analysis
- Fast and scalable backend using FastAPI

---

## Tech Stack

- Python
- FastAPI
- Pandas
- NumPy

---

## Planned API Endpoints

### Get dataset summary
Endpoint:

`GET /summary`

Returns statistical summary of the dataset.

---

### List dataset columns

Endpoint:

`GET /columns`

Returns the column names of the dataset.

---

### Dataset preview

Endpoint:

`GET /preview`

Returns the first few rows of the dataset.
